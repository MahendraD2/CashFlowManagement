# CashVista 1.0

A financial management application built with React and Vite.

## Installation

1. Clone the repository
2. Install dependencies:

\`\`\`bash
npm install
\`\`\`

3. Create a `.env` file in the root directory with the following content:

\`\`\`
VITE_API_BASE_URL=https://api.cashvista.com/v1
\`\`\`

4. Start the development server:

\`\`\`bash
npm run dev
\`\`\`

## API Integration

This application integrates with the CashVista API. Make sure you have the correct API base URL in your `.env` file.

## Dependencies

- React
- React Router
- Axios
- Recharts
- Lucide React
- Tailwind CSS
